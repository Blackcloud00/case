# PIXI-CASE-TEMPLATE
A simple PixiJS + Vite + TypeScript project template.

## Install NodeJS
https://nodejs.org/en/download


## Setup

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm start
```

## After Completing the Project

Once the project is complete, please:

1. **Do not include the `node_modules` folder** in your delivery.
2. Zip the remaining files and folders.
3. Send us the resulting `.zip` file.

Thank you!
